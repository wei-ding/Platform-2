# lib-domain-name-match #

## Build & run ##

```sh
$ git clone git@github.com:Clinical3PO/Stage.git
$ cd Stage/dev/etl/lib-domain-name-match/lib-domain-name-match
$ chmod u+x sbt
$ ./sbt clean && ./sbt compile && ./sbt run
```
or
```
$ ./sbt
> +run
```

## Package ##

```sh
$ ./sbt package
```

lab
drug
