# AWS Auto Scaling for Clinical3PO Paralleling System

## Boto Installation 

### Install via pip:

```sh
pip install boto
```

###Install from source:

```sh
$ git clone git://github.com/boto/boto.git
$ cd boto
$ python setup.py install
```

## Run ##

```sh
$ python aws_autoscaling.py
```
