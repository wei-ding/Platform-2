This createObservation.R program is used to create observation file.

NOTE: Only few sample's of set-a files are provided in this folder.

How To Run:
    1. Download input files from: https://physionet.org/challenge/2012/set-a/, put the files into set-a folder.
	2. Copy ObservationFileCreation folder to R-working directory. (Use getwd() to find R-working directory).
	3. Run  Rscript createObservation.R
